# Java OOP & Collections Practice

This repository contains Java practice programs focused on strengthening understanding of Object-Oriented Programming concepts and Java Collection Framework.

## Topics Covered

### OOP Concepts
- Interfaces
- Abstract Classes
- Anonymous Classes

### Collection Framework
- HashSet
- LinkedHashSet
- TreeSet
- HashMap
- LinkedHashMap
- TreeMap

## Purpose

The purpose of this repository is to:
- Practice and understand core Java concepts
- Learn differences between various collection classes
- Build strong fundamentals for interviews and development

## Key Learnings

- Difference between Interface and Abstract Class
- Use cases of Anonymous Classes
- Ordering and uniqueness behavior in Sets
- Key-Value storage in Maps
- Sorted vs Unsorted collections
- Performance and implementation differences among collection types
